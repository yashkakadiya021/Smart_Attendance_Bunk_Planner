<h1>🎓 Smart Attendance Bunk Planner</h1>

<p>A smart tool that helps students plan their class bunks without falling below the required attendance percentage. Built for hackathons to solve a <b>real student problem</b> with logic, prediction, and a visual dashboard.</p>

<hr/>

<h2>🚀 Features</h2>
<ul>
  <li>✅ Calculates current attendance</li>
  <li>✅ Predicts how many more classes you can bunk</li>
  <li>✅ Warns if you're at risk of falling below required %</li>
  <li>✅ 📈 Plots a graph of bunked classes vs. attendance %</li>
  <li>✅ Easy-to-use Streamlit UI</li>
</ul>

<hr/>

<h2>🧠 How It Works</h2>
<p>
  Based on:
  <ul>
    <li><b>User input</b>: classes attended, conducted, upcoming, and minimum required %</li>
    <li><b>Logic</b>: Calculates safe bunks, projects final attendance</li>
  </ul>
  <i>Uses simple math logic with Python — no heavy ML needed!</i>
</p>
